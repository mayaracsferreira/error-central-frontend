### Executar esse projeto

#### Instalar as bibliotecas
`npm i`

#### Executando o projeto localmente
`ng serve`

Navegar pelo browser até `http://localhost:4200/`

### Compilação do projeto 
`ng build`

O pacote será gerado na pasta `dist/`
